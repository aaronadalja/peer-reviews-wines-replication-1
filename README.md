# Peer Reviews and Wine Demand Replication
data and replication code for 2026 peer reviews and wine demand paper

Author: Aaron Adalja

Date: February 2026

The files contained herein allow for replication of the results for: "Peer reviews and within-category consumer response" by Aaron Adalja, Florine Livat, Bradley Rickard, Alex Susskind

root folder:

peer_reviews_wine_replication.R - This is the R code file that is used to process the data and generate all tables, figures, and statistics in the paper

data folder:

rawdata_pro.csv - raw survey response data
